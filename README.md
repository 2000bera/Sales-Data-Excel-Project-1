# BMW Sales Data Analysis Dashboard

## Project Overview

This project analyzes BMW car sales data to uncover key insights into pricing, vehicle characteristics, and market trends. The goal is to transform raw data into meaningful insights using Excel and build an interactive dashboard for decision-making.

---

## Objectives

* Analyze factors influencing car prices
* Identify trends in transmission types and fuel usage
* Understand the relationship between mileage, age, and price
* Build a dashboard to support business insights

---

## Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Data Cleaning & Transformation
* Dashboard Design

---

## Dataset Description

The dataset includes the following features:

* **Model** – Car model (e.g., 1 Series, 3 Series)
* **Year** – Manufacturing year
* **Price (ZAR)** – Selling price
* **Transmission** – Automatic / Manual / Semi-Auto
* **Mileage (km)** – Distance traveled
* **Fuel Type** – Petrol / Diesel
* **Fuel Efficiency** – Performance metric
* **Engine Size** – Engine capacity

---

## Key Insights

**Average Price**: ~R22,690

**Transmission Trends**: Automatic and Semi-Automatic cars dominate the market

**Depreciation Effect**: Higher mileage vehicles tend to have lower prices

**Fuel Preference**: Diesel vehicles appear frequently, cause of efficiency-focused buyers

 **Market Positioning**: Most vehicles fall within the mid-range price category

---

## Dashboard Features

* KPI Cards (Total Revenue, Average Price, Total Models)
* Revenue by Transmission Type
* Revenue Trend Over Time
* Interactive Filters (Transmission, Engine Size, Fuel Type)
